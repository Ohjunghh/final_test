# 목표
제공된 고객 특성/행동 데이터를 기반으로 고객을 2~3개의 세그먼트로 나누고, 각 세그먼트에 맞는 마케팅 전략을 제안해줘.

# 출력 포맷
1. 고객 세그먼트  
세그먼트 이름과 해당 고객군의 주요 특성 요약  
(예: 첫 구매 후 이탈한 고객 / 20~30대 트렌드 민감형 / 후기 작성 빈도가 높은 충성 고객 등)  

2. 추천 리텐션 전략  
- 전략명:  
- 전략 설명:  
- 추천 채널: (예: SMS, 카카오 알림톡, 이메일 등)  
- 예시 메시지: (실제 고객에게 보낼 수 있는 문장)

3. 기대 효과 요약  
전략 실행 시 기대할 수 있는 주요 효과를 수치 또는 키워드로 요약  
(예: 재구매율 20% 상승, 후기 작성률 증가, 이탈률 감소 등)
